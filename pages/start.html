Getting started

In this article, I will try to get staight to the point of making games. But first, there are a couple things we need to set up.

Installation

Obviously, we need to install Unity itself. You can do that from the official site. I recommend installing the suggested Unity Hub, which helps you to keep track of projects and different versions of Unity.

To make games, we will also need to code. If you've never coded anything before - don't worry. We will go through the basics in the following article. You will, however, need an editor to write the code in. For this I strongly recommend VS Code. It's a lightweight text editor with the option to turn it into a powerful development environment, which helps you with syntax and many other things.
