Getting started

In this article, we will setup everything we need to start making games.

Installation

Obviously, we need to install Unity itself. You can do that from the official site. I recommend installing the suggested Unity Hub, which helps you to keep track of projects and different versions of Unity.

To make games, we will also need to code. If you've never coded anything before - don't worry. We will go through everything step-by-step in the following article. You will, however, need an editor to write the code in. For this I strongly recommend VS Code. It's a lightweight text editor with the option to turn it into a powerful development environment, which helps you with syntax and many other things.

Once you have VS Code installed I also recommend to get the C# extension along with a .NET core 4.7.2 pack from Microsoft. These two paired together will make sure writing code in Unity will be pretty much as good as it gets. If you need some extra help, you can follow this article.
